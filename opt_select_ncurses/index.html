opt_select_ncurses

Turn any CLI output (git, grep, ls, etc.) into an interactive selector with Vim-style navigation. 

generate → select → perform action

Demo

Use cases

  • Select Git branches
  • Pick files and run actions
  • Work with modified files
  • Search and jump to results

Install

git clone https://github.com/shyjun/opt_select_ncurses.git
cd opt_select_ncurses
make
sudo make install
